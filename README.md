# HyperCompute MVP

## 一键安装并运行

### Linux / macOS

```bash
bash scripts/one_click_install_and_run.sh
```

> 轻量模式（更少服务）：

```bash
bash scripts/one_click_install_and_run.sh --lite
```

### macOS 双击运行

仓库根目录提供了 `一键安装并运行.command`，双击后会自动执行安装与部署脚本。

## 常用命令

```bash
make logs    # 查看日志
make down    # 停止服务
make reset   # 重置环境
```
